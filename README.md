# Video Splitter

## 概要

このツールは、Python と customtkinter を用いて作成されたGUIアプリケーションです。動画ファイル（mp4, avi, mov, mkv など）を複数の区間に分割し、指定したフォルダに保存できます。

## 主な機能

- 動画ファイルの読み込み・プレビュー
- シークバーによるフレーム移動・ズーム
- 分割区間（開始・終了ポイント）の設定
- 区間リストの編集（タイトル・開始/終了時刻の編集、削除）
- 分割リストの保存・読み込み（プロジェクトファイル）
- 指定フォルダへの一括出力

## 使い方

1. **動画ファイルを選択**
   - 「動画ファイルを選択」ボタンから動画を読み込みます。
2. **分割区間の設定**
   - シークバーやフレーム操作ボタンで分割したい開始位置に移動し、「開始ポイント設定」ボタンを押します。
   - 終了位置に移動し、「終了ポイント設定（分割追加）」ボタンを押すとリストに追加されます。
3. **区間リストの編集**
   - タイトルや開始/終了時刻はリスト上で直接編集できます。
   - 不要な区間は「×」ボタンで削除できます。
4. **出力先フォルダの選択**
   - 「フォルダを選択」ボタンで出力先を指定します。
5. **分割実行**
   - 「分割実行」ボタンで、リストの全区間を一括で動画ファイルとして出力します。
6. **プロジェクトの保存・読み込み**
   - 区間リストや出力先を保存/復元できます。

## 必要なライブラリ

- Python 3.8 以降
- customtkinter
- tkinter
- opencv-python
- pillow

`uv` を使用の場合は、以下のコマンドで依存関係をインストールできます。

```
uv sync
```

## 起動方法
### `uv` を使用の場合
```
uv run python main.py
```

## ファイル構成

- `main.py` : アプリ本体
- `pyproject.toml` : 依存関係管理
- `uv.lock` : ロックファイル（依存関係固定用）
- `README.md` : このファイル

## 注意事項

- 大きな動画ファイルの分割には時間がかかる場合があります。
- サポートされる動画形式は OpenCV の仕様に依存します。
